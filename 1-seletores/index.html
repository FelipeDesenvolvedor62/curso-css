SELETORES

Os seletores são responsável por selecionar Tags, Classes, Id e atributos para adicionar a estilização


Seletor UNIVERSAL

Aplica o css em todos elementos da pagína

* {

background-color: rgb(223, 182, 182);

}

Seletor por TAG

Aplica o css em todas as tags

h1 {

font-size: 50px;

}

Seletor por CLASSE

Aplica o css para todas as classes

.seletor1 {

color: green;

}

Seletor por ID

Aplica o css para todos os id

#seletor2 {

color: blue;

}

Seletor por ATRIBUTO

Aplica para todos os atributos

[type="text"] {

border-radius: 50px;

}

Seletor por PSEUDO

Aplica para todos os atributos/classes com ações

[type="text"]:focus {

border: solid 5px green;

}
